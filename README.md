# komodo
Para ejecutar el proyecto, debe tenerse una instalación de docker en el sistema.
Una vez cumplidos los pasos anteriores, deberá abrirse el directorio komodo en consola y ejecutar el comando "docker-compose up". A partir de este momento, el programa podrá iniciarse en cualquier momento usando el mismo comando. Es posible que la primera ejecución fracase, debido al tiempo de creación de la base de datos. En caso de que esto suceda, solamente será necesario ejecutarlo una segunda vez.
